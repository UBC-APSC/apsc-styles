# File Structure

```bash
├── apsc-scss
├── dev
└── dist
```

- ```apsc-sccs``` contains all of the source SCSS files
- ```dev``` contains a static boilerplate HTML page for basic element and component styling
- ```dist``` contains minified production CSS files
